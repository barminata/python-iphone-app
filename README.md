# Учим Python

MVP мобильного веб-приложения для обучения Python с iPhone.

## Что уже есть

- React + TypeScript в браузере без backend
- Tailwind CSS через CDN
- Запуск Python через Skulpt
- 5 моковых уроков
- Сохранение прогресса, достижений и кода в `localStorage`
- Mobile-first интерфейс для iPhone

## Как открыть у себя локально

Открой папку `python-iphone-app` через любой простой локальный сервер.

Например, если на компьютере есть Python:

```bash
python3 -m http.server 4173
```

Если сервер запущен прямо из папки `python-iphone-app`, адрес будет:

`http://localhost:4173`

Если сервер запущен из другой папки, открой в браузере путь до каталога, где лежит `index.html`.

## Как сделать, чтобы приложение работало на другом телефоне само по себе

Важно: сайт не может сам по себе жить "внутри" чужого iPhone, если он не опубликован в интернете или не упакован как нативное приложение. Для первого этапа здесь уже подготовлен самый простой путь: это статический PWA-сайт, который можно выложить на хостинг без backend.

Что уже добавлено:

- `manifest.webmanifest` для установки на экран телефона
- `sw.js` для кеширования и более удобной работы как PWA
- иконки приложения

Самый простой вариант публикации:

1. Зайди на [Netlify Drop](https://app.netlify.com/drop)
2. Перетащи туда папку `python-iphone-app`
3. Дождись, пока сервис даст публичную ссылку вида `https://...netlify.app`
4. Отправь эту ссылку сестре
5. На iPhone она сможет открыть сайт и при желании добавить его на главный экран через Safari

## Как опубликовать через GitHub Pages

1. Убедись, что репозиторий уже загружен на GitHub
2. В настройках репозитория открой `Pages`
3. В `Source` выбери `Deploy from a branch`
4. Выбери ветку `main` и папку `/ (root)`
5. Нажми `Save`

Сайт обычно появляется через несколько минут по адресу:

`https://<твой-логин>.github.io/<имя-репозитория>/`

Альтернатива:

- GitHub Pages
- Cloudflare Pages
- Vercel

## Как установить на iPhone

1. Открыть опубликованную ссылку в Safari
2. Нажать "Поделиться"
3. Выбрать "На экран Домой"
4. Запустить как обычное приложение
